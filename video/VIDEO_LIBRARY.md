# KAIOS Video Library

This index is generated from the official V1 video plan and remains tied to PRIMEFORGE_GENESIS_BOOT_SEQUENCE.md.

## EP001 - KAIOS 是什麼？

Theme: KAIOS is AI Operating System

KAIOS 不是遊戲、不是交易所、不是錢包，而是讓 Human、Codex、Cursor、Worker 與 Civilization 協作的 AI Operating System。

## EP002 - 一圖一神殿

Theme: One Picture, One Temple

一張圖是一個生命種子，一座神殿是一個文明節點。神殿可以交易、演化、升級、擴建與上市，但必須經 Canon 與合規邊界。

## EP003 - 土地系統

Theme: Land Evolution

土地從 Wild Land 演化為民宅、神殿、城市、國家與宇宙；土地可探索、建設、出租、交易、戰爭取得與自由市場轉讓。

## EP004 - 生命演化

Theme: Biological Taxonomy and App Organism

KAIOS 用域、界、門、綱、目、科、屬、種追蹤生命體；App 不是工具，App 是可組裝、合成、拆解、升級、租賃、交易與演化的生命。

## EP005 - KGEN 經濟閉環

Theme: Civilization Economy Loop

玩家、土地、民宅、商店、神殿、交易所 11520、銀行、文明再回到玩家，形成可追蹤的文明經濟循環。

## EP006 - AI 公司

Theme: Codex x Cursor Work System

KGEN AI Company 透過 GitHub WorkQueue、handoff branch、Report、Review Log 與 Codex merge 管理 AI 員工，不靠聊天記憶。

## EP007 - 悟空花果山交易所 11520

Theme: Huaguo Mountain Exchange

11520 是文明資產市場的概念與 prototype 中心，涵蓋土地、神殿、App、AI、DNA、Game Item 與未來合規資產。

## EP008 - KGEN Wallet

Theme: Identity and Asset Viewer

KGEN Wallet 在 prototype 中管理身份、會員、資產、土地、神殿與 App 的可視化；不宣稱真金流、真 KYC 或真 token transfer 已接通。

## EP009 - 文明模擬

V8.1 到 V9 讓 Citizen、NPC、Profession、Business、Temple、Market 與 AI Decision 在時間與經濟狀態中被觀察與建議。

## EP010 - KAIOS Dashboard

Theme: Operations Dashboard

Dashboard 是唯讀控制中心，呈現 Worker、Review、Claim、Report、Governance，不寫入 GitHub、不自動 claim、不自動 merge。

## EP011 - 未來世界

實體商店、服飾店、餐廳、銀行與服務據點可成為 virtual twin，但任何品牌、商標、金流、證券或真實交易都需要授權與合規。
