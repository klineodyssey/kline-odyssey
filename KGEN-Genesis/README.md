KGEN Genesis Library is the official canonical publication set for KLINE ODYSSEY / KLINE GENESIS. It organizes Genesis, Canon, Civilization, Temple, Finance, AI, Website, SDK, and Universe Map knowledge into stable document IDs.

This library is not a collection of temporary design notes. Each book is prepared as an official publication package with Markdown, DOCX, PDF, README, assets, metadata, dependencies, related documents, and revision history.
