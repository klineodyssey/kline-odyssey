# K線5D峽谷推塔戰 V0.1

## 概述

K線5D峽谷推塔戰是 KGEN Universe 的原創 MOBA 概念遊戲原型，以 K 線金融戰場為世界觀，純前端 HTML/CSS/JS，不依賴任何框架或外部 CDN。

## 設計理念

- **K線 = 戰場**：K 線漲跌決定峽谷戰線推進方向
- **多方陣營（Bull Faction）**：從 12345 悟空財神殿出發，做多時推進
- **空方陣營（Bear Faction）**：從 16888 廣寒宮出發，做空時反撲
- **中路節點**：11520 花果山交易所、18888 靈霄寶殿、18921 斬妖台為中立可佔領節點
- **KGEN 宇宙觀**：宇宙物理法則、K軸作功、多空推塔

## 遊戲玩法（Prototype V0.1）

1. **做多**（📈）：多方增加兵線，K線漲時自動推進空方塔
2. **做空**（📉）：空方增加兵線，K線跌時自動反撲多方塔
3. **持倉**（⏸️）：修復自方塔 HP
4. **Approve**（✅）：解鎖宇宙節點加成
5. **發財金**（🧧）：多方大招，對空方塔造成大量傷害
6. **轉日呼吸**（🌬️）：空方大招，對多方塔造成大量傷害
7. **佔領節點**：點擊地圖節點（11520/18888/18921）佔領，獲得加成
