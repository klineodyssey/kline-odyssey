## 二、KGEN 是什麼？

**KGEN（K-Line Genesis Energy Node）**  
是一種代表「存在質量」的鏈上原生資產。

它不是遊戲幣，  
不是短線炒作標的，  
而是整個 KLINE 宇宙的**基礎能量單位**。

### KGEN 的核心定位

- 質量單位（Existence Weight）
- 治理權重（Governance Power）
- 分潤憑證（Reward Share）
- 系統能量（System Fuel）

**持有即存在，存在即有位置。**

**KLINE App Game** 不是娛樂遊戲，

在這裡：

- 操作不是按鈕，是紀律  
- 升級不是課金，是績效  
- 角色不是設定，是行為紀錄
