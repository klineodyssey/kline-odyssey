## 8. 保證金生命機制

### 8.1 保證金 = 血量

```text
Margin = Blood
```

當 App 生命被交易、抵押、借貸或用於策略運算時，保證金就是生命血量。

### 8.2 血量階段

```text
100%：正常運行
70%：進入節能模式
50%：出售非核心器官
30%：關閉高耗能 App
10%：進入逃生模式
0%：解體，回到走路取經
```

### 8.3 自動賣出器官

當保證金下降時：

```text
先賣裝飾器官
再賣輔助器官
再賣高耗能器官
最後保留核心 DNA
```

例：

```text
音響 App → 可先出售
地圖 App → 保留到中後期
錢包 App → 核心器官不可先賣
Autopilot → 最後保留
DNA / README → 永不刪除
```

### 8.4 虧光回到走路取經

當保證金歸零：

```text
變形金鋼解體
App 器官全部下架
交易能力關閉
只保留走路取經模式
```

走路取經模式代表：

```text
不交易
不槓桿
不自動買賣
只靠探索、任務、修行、點燈、還願重新累積生命能量
```
