# Revision Runtime

**Runtime ID:** KAIOS-V9.1-RUNTIME-REVISION  
**Status:** Prototype  
**Mode:** Codex-gated.

The Revision Runtime keeps a useful DRAFT alive while requiring corrections.

## Common Revision Requests

- Unique report path.
- Measurable acceptance criteria.
- Narrower task scope.
- Explicit legal or security boundary.
- Clearer dependencies.

## Output

The output is a revision request that moves the task to `NEEDS_REVISION`.
