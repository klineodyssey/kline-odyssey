# KAIOS V9.1 Read-Only WorkOrder Review Dashboard

This dashboard is a static GitHub Pages interface for the V9.1 AI WorkOrder Review Loop.

It displays:

- Draft WorkOrders
- Under Review
- Needs Revision
- Approved for Open
- Rejected
- Blocked
- Archived
- Risk Distribution
- Human Review Required
- Duplicate Alerts
- Dependency Alerts
- Codex Review History

The dashboard is read-only. It does not write to GitHub, does not use a token, does not promote WorkOrders and does not modify the WorkQueue.
