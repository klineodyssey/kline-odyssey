# KAIOS V8.1 AI Runtime

## Definition

AI Runtime defines how AI entities attach to Citizen, Temple, App, NPC, Business, Market or Civilization records. AI is a life organ and runtime assistant. AI may plan, validate, route, summarize and detect missing relationships.

## Canon Lock

AI records must default to `canon_lock: true`. AI cannot reverse Canon, modify Boot, rewrite Runtime CURRENT, or push protected path changes without explicit review and permission.

## Allowed Actions

- read Universe Graph snapshots
- detect missing parent/child relations
- propose WorkOrders
- summarize citizen/profession/economy gaps
- flag legal, security, and regulated boundary risk
- generate reports for Codex review

## Disallowed Actions

- claim real-world partnership without evidence
- create financial license claims
- execute trade or payment
- write to protected paths
- bypass Codex review
