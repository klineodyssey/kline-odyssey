# KAIOS V8.1 Economy Runtime

## Definition

Economy Runtime connects Profession output to Business, Market, Listing, Transaction, Exchange, Bank and Governance. It turns citizen work into civilization economy data.

## Runtime Flow

```text
Citizen Work -> Business Output -> Market -> Listing -> Transaction -> Governance -> Civilization Metric
```

## Supported Entities

- Business
- Exchange
- Bank
- Market
- Listing
- Transaction
- Governance
- Item
- Mission
- Quest

## Bank Boundary

Bank in V8.1 remains simulation unless a future production module integrates licensed financial institutions and local compliance. Bank records may model reserve, credit, risk and audit, but must not claim real banking authority.

## Exchange Boundary

Huaguo Mountain Exchange 11520 remains a KGEN runtime model. Regulated asset trading requires future legal authorization and compliance review.
