# KAIOS V9.0 Civilization AI Dashboard

**Path:** `KGEN-KAIOS/V9.0/dashboard/`  
**Public URL:** https://klineodyssey.github.io/kline-odyssey/KGEN-KAIOS/V9.0/dashboard/

## Purpose

The V9.0 dashboard displays AI decisions, risk alerts, recommended Draft WorkOrders, civilization health, economy signals, temple signals, citizen signals, Human Overrides and Codex Reviews in read-only mode.

## Boundary

The dashboard does not write to GitHub, does not execute tasks, does not merge branches, does not deploy, does not transfer tokens and does not use a GitHub token.
