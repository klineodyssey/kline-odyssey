# KAIOS V8.2 Read-Only Economy Dashboard

**Path:** `KGEN-KAIOS/V8.2/dashboard/`  
**Public URL:** https://klineodyssey.github.io/kline-odyssey/KGEN-KAIOS/V8.2/dashboard/

## Purpose

This dashboard displays the V8.2 Civilization Economy Engine in read-only mode. It reads public JSON examples, schema files and release reports from the repository. It does not write to GitHub, does not claim tasks, does not merge branches, does not connect wallets and does not execute trades.

## Views

- System Health.
- Temple Economy.
- Citizen Economy.
- Market Overview.
- Business Ranking.
- Resource Flow.
- Civilization Health.
- Alerts.
- Next Recommended Task.

## Data Sources

The dashboard configuration is stored in `dashboard.config.json`. All runtime data sources are under `../examples/` and `../schemas/`.

## Boundary

All bank, market, exchange, investment, lending, interest, NFT, token and listing references are Concept / Prototype / Simulation unless a future governed production release promotes them after legal, security, governance and compliance review.
