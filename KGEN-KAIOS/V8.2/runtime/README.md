# KAIOS V8.2 Runtime Documents

This directory defines how the Civilization Economy Engine reads V8.1 Universe Data Layer entities and turns them into economy signals, simulated business state and governance inputs.

## Runtime Documents

| File | Purpose |
|---|---|
| `CIVILIZATION_ECONOMY_RUNTIME.md` | Full economy loop from Temple to Civilization Growth. |
| `BUSINESS_RUNTIME.md` | Business lifecycle, employment, production, revenue and upgrade logic. |
| `MARKET_RUNTIME.md` | Market operations and price discovery simulation. |
| `BANK_RUNTIME.md` | Treasury, reserve and bank simulation boundary. |
| `EXCHANGE_RUNTIME.md` | Huaguo Mountain Exchange 11520 and asset marketplace runtime. |
| `GOVERNANCE_SIGNAL_RUNTIME.md` | GDP, population, employment, activity and health signal derivation. |

All runtime files are Concept / Prototype / Simulation unless explicitly promoted by future legal, security, governance and production review.
