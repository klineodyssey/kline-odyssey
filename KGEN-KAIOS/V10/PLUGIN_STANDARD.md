# Plugin Standard

**Document ID:** KAIOS-V10-PLUGIN-STANDARD  
**Version:** V10.0  
**Status:** Runtime Standard

Plugin Runtime allows Temple, Business and App surfaces to extend functionality without duplicating the core system.

## Plugin Fields

- plugin_id
- host_type
- host_id
- owner
- permissions
- input_schema
- output_schema
- install_state
- version
- risk_level
- audit_events

## Install Rule

Any plugin that touches wallet, payment, membership, marketplace, GitHub, AI dispatch, protected path or blockchain state requires Codex Review before activation.
