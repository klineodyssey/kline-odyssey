# KAIOS V10 Operating Dashboard

This dashboard is read-only. It visualizes KOS system status, service status, wallet status, membership status, marketplace status, AI Company status, temple status, bank status and blockchain status.

It does not:

- authenticate real users
- create real memberships
- sign wallet messages
- process payments
- transfer tokens
- deploy contracts
- write to GitHub

Production URL:

`https://klineodyssey.github.io/kline-odyssey/KGEN-KAIOS/V10/dashboard/`
