# Role Capability Profiles

## 1. Profile, Not Caste

A `RoleCapabilityProfile` is a curated atom selection plus evidence, maintenance and context requirements. It is not a species, bloodline, legal title, immutable identity or exclusive seat. A role title never grants atoms by itself.

## 2. Human Canon Anchors

| Anchor | Formal interpretation | Distinguishing design |
|---|---|---|
| Zhu Bajie / 36 | `GA_COUNT_36`, a curated Tiangang-style versatile profile | Competent life, action, learning, production, social and adaptive decision subset; 36 changes represent bounded transformation breadth, not a low moral rank |
| Sun Wukong / 72 | `GA_COUNT_72`, a high-adaptation role profile | Broader perception, learning, reasoning, engineering and transformation interoperability; 72 changes are a reference profile, not a permanent cap on every Wukong individual |
| Yudi / 107 | `GA_COUNT_107`, all canonical atoms except GA108 | Celestial governance synthesis is complete, but final whole-system integration closure is absent |
| Rulai / 108 | `GA_COUNT_108`, all canonical atoms including GA108 | Complete awareness, coordination, restraint and Genesis whole-system integration; no constitutional, legal or real-asset supremacy |

GA36 and GA72 profiles may select atoms across all relevant domains; they are not required to be the first 36 or 72 catalog IDs. Yudi's canonical reference profile is explicitly all except GA108 so the 107/108 difference is deterministic.

## 3. Ordinary Humans

Humans use Species Genome inherent capabilities plus variable individual atom profiles. There is no fixed atom count for a mortal, citizen, profession or family. Evidence, training, environment and chosen life path determine the profile.

## 4. Zodiac and Classical Archetypes

The twelve zodiac signs use optional, variable `CULTURAL_ROLE_PROFILE` combinations, not fixed counts or inherited destiny. Bull Demon is preferably `BULL_DEMON_ARCHETYPE` or a project-specific classical role profile, not a separate species unless a later taxonomy review proves species-level differences.

| Reference | Recommended model | Atom rule |
|---|---|---|
| Bull Demon | Classical `BULL_DEMON_ARCHETYPE` capability/role profile | Variable reviewed profile; no number is invented here |
| Sha Wujing | Service, endurance, logistics and stewardship role profile | Variable reviewed profile; no permanent count |
| Dragon lineage | Species/Family/Genus only when an approved taxonomy defines stable lineage; role profile otherwise | Species-compatible variable atom set |
| Twelve zodiac | Cultural archetype overlay, never deterministic biology or destiny | Variable opt-in combinations |
| Ordinary mortal | Species Genome plus individual capability profile | Evidence-derived variable count |

## 5. Species, Role, Skin and Adapter

| Concept | Correct use |
|---|---|
| Species | Stable biological/digital lineage with body and Species OS contract |
| Role | Mission, profession, title or narrative responsibility |
| Skin | Presentation only; grants no capability |
| Capability Profile | Verified atom selection and context |
| Licensed Fiction Adapter | Explicitly licensed mapping for protected external IP |

Modern protected characters are not official saleable KAIOS roles without a license. Use generic archetypes such as `TRANSFORMING_MECHA`, `SOLAR_SUPERHUMAN` and `BULL_DEMON_ARCHETYPE`; do not copy protected names, marks, appearance, lore, dialogue or signature assets.

For classification purposes, a transforming robot or solar-powered superhuman concept is a `GENERIC_ARCHETYPE` plus capability profile. A specific modern franchise character is a `LICENSED_FICTION_ADAPTER` only after rights verification. A skin remains visual and grants no atoms.

## 6. Fair Access to GA107/108

Any eligible player, AI, NPC or life may pursue the same public eligibility path. GA107/108 require all dependency, evidence, maintenance, balance, safety and review gates. They cannot be purchased, inherited automatically or reserved for Father/Human/one provider.

GA108 should be reachable but intentionally rare and long-term. It remains revocable/suspendable when evidence or integrity fails and never overrides Human Final Authority.
