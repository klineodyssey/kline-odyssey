# DNA Integrity and Balance

## 1. Integrity Checks

Every revision checks:

- unauthorized mutation;
- runaway growth;
- resource overconsumption;
- organ incompatibility;
- species mismatch;
- circular dependency;
- corrupt expression;
- unbounded reproduction;
- economic exploit;
- hidden privilege;
- malicious capability;
- version mismatch;
- missing evidence;
- secret or cross-tenant data inheritance.

## 2. Uncontrolled Cell Growth Pattern

`UNCONTROLLED_CELL_GROWTH_PATTERN` is a digital-life integrity and game-disease abstraction: a cell/module replicates, allocates resources or expands authority beyond declared bounds. It is not a medical diagnosis and provides no real treatment instruction.

Detection signals may include repeated unbounded spawn requests, quota bypass, missing termination, resource divergence, unauthorized dependency creation or lineage loops. Response states are `QUARANTINED`, `SUSPENDED`, `ROLLBACK_REQUIRED`, `HUMAN_REVIEW_REQUIRED` or `ARCHIVED`.

## 3. Balance Dimensions

| Dimension | Required controls |
|---|---|
| Species | Trait limits, body compatibility, welfare/safety envelope |
| Energy/resource | Conservation budget, input accounting, maintenance cost |
| Environment | Water, climate, land, waste and ecological impact |
| Population | Carrying capacity, reproduction cap, generation spacing |
| Economy | Supply/demand stress, anti-inflation, price/royalty transparency |
| Mission/game | Cooldown, diminishing returns, task purity, anti-cheat |
| Security | Least privilege, sandbox, tenant isolation, secret boundary |
| Governance | Evidence, review, appeal, Human override and rollback |

## 4. Risk Classes

- `R0`: descriptive/public metadata.
- `R1`: low-impact bounded capability.
- `R2`: operational, production, movement or resource capability.
- `R3`: market, social, reproduction or cross-system capability.
- `R4`: governance, orbital or broad multi-agent capability.
- `R5`: whole-system integration or irreversible/high-impact proposal; Human Final Approval required.

## 5. Fail Closed

Missing evidence, unknown module, checksum mismatch, unavailable dependency, stale test, unbounded parameter, hidden privilege, private-data leak or ambiguous GA semantics blocks activation. A failed candidate remains evidence; it is not silently repaired in production.

## 6. No Real Biology

All species tuning is simulation architecture. No real gene editing, cultivation protocol, pathogen work, weaponization or clinical advice is authorized.
