# KAIOS V8.3 Time Dashboard

**Path:** `KGEN-KAIOS/V8.3/dashboard/`  
**Public URL:** https://klineodyssey.github.io/kline-odyssey/KGEN-KAIOS/V8.3/dashboard/

## Purpose

The V8.3 dashboard displays the Civilization Time Engine in read-only mode. It reads the V8.3 examples and schemas from public repository files. It does not write to GitHub, does not advance a production clock, does not claim tasks, does not merge branches and does not connect to wallets, banks or exchanges.

## Panels

- World Clock.
- Tick Status.
- Timeline Health.
- Citizen Behavior.
- Business Behavior.
- Temple Activity.
- Event Engine.
- Governance Response.

## Boundary

All records are Concept / Prototype / Simulation. War, disaster, banking, exchange, market and governance terms are simulation language only.
