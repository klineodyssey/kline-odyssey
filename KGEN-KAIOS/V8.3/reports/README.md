# KAIOS V8.3 Reports

This directory stores QA and release records for the Civilization Time Engine.

## Reports

| File | Purpose |
|---|---|
| `KAIOS_V8_3_QA_REPORT.md` | Validation record for schemas, examples, viewers, read-only boundary and protected paths. |
| `KAIOS_V8_3_RELEASE_REPORT.md` | Release summary for V8.3, including counts, URLs and V9.0 recommendation. |

Reports are documentation artifacts only. They do not advance the simulation, modify WorkQueue, create handoff branches or touch protected systems.
