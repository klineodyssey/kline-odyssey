## Purpose

Defines K280 / Land Runtime V1 architecture for parcel geometry, ownership rights separation, territory zones, residence protection, and lunar land registry. This package is **architecture documentation only**.

## Non-Goals (NOT AUTHORIZED)

- Implementation Planning / WorkQueue / Runtime Implementation
- Real KGEN payment, real land trade, Land NFT
- Real financial or military services
- Modification of Runtime CURRENT or Universe Map

## Relation to Civilization Economy Runtime

Civilization Economy Runtime V1 **must cite** this Land Runtime and **must not** redefine K280, Parcel, Coordinate, or Land Ownership Geometry.
