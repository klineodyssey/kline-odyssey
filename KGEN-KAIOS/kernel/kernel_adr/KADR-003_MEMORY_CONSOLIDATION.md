# KADR-003: Memory Consolidation

## Context

V1 已有 Working Memory、正式 source priority、checkpoint 與 reviewed summary，但 Long Memory、Civilization Memory、Reflection 與 Dream 尚未形成獨立契約。

## Decision

V1 必須採 deterministic consolidation：

1. 只從 reviewed evidence 產生 summary。
2. 保留 source、hash、review ID 與 supersession。
3. 不以 confidence 取代 grounding。
4. 不在 Sleep 自動改 Canon、Mission、Skill 或權限。

## Deferred

- V1.1：Long / Civilization Memory retrieval、contradiction policy、structured reflection。
- V2：Dream sandbox。

## Rejected

Dream 不得在 V1 Sleep 內自動建立正式 memory、code、Mission 或 mutation。

## Consequences

Memory 成長較慢，但可追溯且不會因不可見推理污染正式狀態。
