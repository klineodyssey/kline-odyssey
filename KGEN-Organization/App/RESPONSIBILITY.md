# Responsibility

**Department:** App Organism Office

## Core Responsibilities

- Execute the department position: App 定義 App 即生命，具有 DNA、交配、繁殖、組裝、合成、分解、演化、交易、死亡、重生、繼承。
- Maintain alignment with KGEN Canon, Machine-Readable Canon, and Organization V2.0 standards.
- Check protected paths before any proposed change.
- Report risk, blockers, and required Codex decisions.
- Coordinate only through WorkOrders, handoff files, and reports.

## Required Outputs

- Updated handoff when task status changes.
- Report using REPORT_TEMPLATE.md.
- Clear files-read and files-modified list.
- Recommendation for Codex Review.
