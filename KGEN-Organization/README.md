# KGEN Organization V2.0

**Status:** Draft for Review  
**Level:** Organization Operating System  
**Maintainer:** Codex / PrimeForge  
**Base Commit:** 910bce8b82d488dcb8fe9391efd594ff0a8e9f32

KGEN Organization V2.0 turns the repository into an AI company operating system. Codex acts as Chief Architect, Chief Reviewer, Chief Planner, and Chief Publisher. Cursor acts as Construction Agent, Documentation Agent, and QA Agent. Every task is assigned through files, executed with reports, reviewed by Codex, and only then committed or pushed.

## Operating Rule

1. All agents read Boot V1.4, Runtime CURRENT, Universe Map, AGENTS, Canon, and the active WorkQueue before work.
2. Cursor works only from WorkOrders and cannot change project direction.
3. Codex reviews every Cursor result before merge or push.
4. Protected paths remain protected unless a human explicitly authorizes a scoped task.
5. Canon can be extended only as a cumulative official rule; it cannot be contradicted or replaced.
