# KGEN Organization Reports

All Organization V2.0 reports are stored here.

## Report Rule

Cursor writes reports here only after an assigned WorkOrder. Codex reviews each report before commit, merge, or push. Reports must include files read, files modified, checks run, risks, blocked items, and recommendation.

## Naming

Use the WorkOrder ID as prefix, for example:

`ORG-P2-018_QA_VALIDATION.md`

## Protected Systems

Reports may mention protected paths, but they must not modify contracts, `K線西遊記/temples/12345`, wallet, bridge, Boot, Runtime CURRENT, or final-whitepaper.
