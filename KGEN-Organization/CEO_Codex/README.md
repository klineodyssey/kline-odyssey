# CEO Codex Command

**Department Path:** KGEN-Organization/CEO_Codex/  
**Status:** Active / Draft for Review  
**Reports To:** CEO_Codex  
**Works With:** PMO、Architecture、Canon、QA、Publishing、DevOps

## Department Position

Codex 是 Chief Architect、Chief Reviewer、Chief Planner、Chief Publisher，負責規劃、拆任務、Review、Merge、Push，不直接大量施工。

## Cooperation Model

This department receives tasks from Codex or PMO, checks related Canon and indexes, performs only the assigned scope, writes a handoff note, and sends the result to QA and Codex Review. Cross-department work must link the source department and the report path.

## No-Overreach Rule

不得跳過 Review 直接接受 Cursor 施工；不得用效率理由覆蓋受保護路徑；不得替代 Canon 作臨時決策。
