## 三、KGEN 的核心功能

KGEN 不代表報酬，  
而代表 **你在系統中的權重與存在感**。

### 1️⃣ 系統權重單位
- 影響治理投票權
- 影響資源分配順序
- 影響角色／模組解鎖層級

### 2️⃣ 行為門檻機制
- 高階功能需達到最低 KGEN 門檻
- 防止濫用、機器行為與短期投機

### 3️⃣ 能量消耗與燃燒
- 使用系統資源時消耗 KGEN
- 部分 KGEN 會被永久燃燒
- 確保長期稀缺與動態平衡

## 四、KGEN 與「收益」的關係（重要澄清）

KGEN **不是分紅權本身**，  
而是「取得分配資格的門票」。

### 系統中可能產生的價值來源：
- 交易稅分配池
- 遊戲內經濟活動
- IP 授權與內容收入
- 社群與應用服務收益

### KGEN 的角色是：
> 決定你是否能參與、以及參與的「權重比例」

是否產生收益，取決於：
- 系統是否運行
- 模組是否產生現金流
- 治理是否通過分配機制

KGEN 交易採 **極簡、低干擾設計**：

- 轉帳：免稅
- 交易稅：0.3%

### 交易稅分配結構（示意）
- 0.1% → 系統黑洞（永久燃燒）
- 0.1% → 系統金庫（Bank）
- 0.05% → 流動性池（LP）
- 0.05% → 生態回饋池（Reward）

> 稅制目標不是抽取，
> 而是維持系統穩定與長期存活。
