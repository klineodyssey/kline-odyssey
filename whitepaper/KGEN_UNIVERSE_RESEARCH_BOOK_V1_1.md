## 3. 黑洞（Burn Pool × LP Pool）質量模型
黑洞質量 = 系統內可吸收的最大能量

### 黑洞由三部分組成：
1. 銷毀池（Burn）  
2. 流動性池（LP）  
3. 遊戲消耗池（Game Sink）

### 黑洞質量公式：
```
M_blackhole = Burn + LP * C + GameSink
```
C = 流動性參數（0.3–1.0）

### 黑洞壽命：
黑洞質量越大 → 壽命越長  
黑洞質量越小 → 易蒸發、易「宇宙坍縮」

---

## 4. 白洞（Natural Recycle）不是增發
白洞 = 再循環器  
不是印新幣  
不是總量增加  

來源：
- Burn 的霍金蒸發  
- 遊戲消耗碎片  
- 稅池回收碎量  
- Fuel 池壓力釋放  
- 行為粒子化（行為 → 代幣碎片 → 再合成）

白洞釋放量 R：
```
R ≤ (M_blackhole × k)
```
k 約為 0.0001~0.01  

白洞釋放必須永遠小於黑洞吸收能力。

---

## 5. 霍金輻射（Hawking Radiation）公式（KGEN 版本）
實體黑洞的蒸發公式：
```
蒸發率 ∝ 1 / (質量^2)
```

在 KGEN 宇宙中：
```
H_rate = α / (M_blackhole^2)
```
α = 宇宙調諧常數  
M_blackhole 越小 → 蒸發越快 → 幣價不穩  
M_blackhole 越大 → 蒸發越慢 → 宇宙長壽

## 6. Autopilot 宇宙流（Flux）
AI 自動駕駛負責管理：
- 白洞釋放（碎片掉落率）  
- 黑洞吸收（燒毀 ROI）  
- 時間事件（跨年、悟空誕生日）  
- 價格事件（ATR×72突破、宇宙遷移）  

宇宙流動方程式：
```
Flux = (WhiteFlow - BlackFlow) + ExternalEnergy
```

ExternalEnergy = 市場注入的外部資金（真實世界）

Flux 必須接近 0 → 宇宙穩定。

---

## 7. 宇宙穩定條件（核心公式）
宇宙不坍縮條件：
```
WhiteFlow ≤ BlackFlow + GameSink
```

宇宙不爆炸條件：
```
BlackFlow ≤ ExternalEnergy × ψ
```
ψ = 市場吸收係數（0.1–0.5）

---

## 8. 宇宙坍縮（Crash）＝ 黑洞蒸發至 0
條件：
- LP 被完全抽乾  
- Burn 池蒸發殆盡  
- GameSink 未形成  

這代表：
```
M_blackhole → 0
```
母船墜機。
