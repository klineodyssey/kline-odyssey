## 1. KGEN 是什麼？
KGEN 是《KLINE Odyssey》宇宙中的核心能量單位，用於：
- 存在質量（Mass / Existence Weight）
- 系統燃料（System Fuel）
- 分潤憑證（Reward Share）
- 治理權重（Governance Power）

核心法則：
沒有質量，就沒有位置。

500 年是文明週期（Epoch）：
回流、分配、建設在同一套制度中持續運作，並在紀元結束時觸發制度交接。
